"# RGP-game"
<h2>RPG GAME</h2>
<p>
Сражения персонажей.
Тут будет описание игры.
</p>

<h3>Сделано</h3>
<ol>
  <li>Параметры юнитов: Хит-поинты, Сила, Ловкость, Инициатива</li>
  <li>Айтемы: Оружие, Броня</li>
  <li>Механика боя: Уклонение, Критические удары</li>
  <li>Готовые юниты: Солдат, тяжелый солдат, орк, огр, дворф</li>
  <li>8 видов оружия</li>
  <li>4 вида брони</li>
</ol>

<h3>Планы</h3>
<ol>
  <li><del>Вывод статистики боя на фрейм</del></li>
  <li><del>Фрейм создания персонажа, выбора оружия и защиты</del></li>
  <li>Сделать систему усталости (Сила станет важнее)</li>
  <li>Добавить виды урона (Колющий, режущий, оглушающий)</li>
  <li>Эффекты (кровотечение, оглушение)</li>
  <li>Опыт, система уровней</li>
  <li>Локации, перемещение по локациям</li>
</ol>
